## FaceITBot's Privacy Policy

### By using the FaceITBot you accept with its privacy policy. 

### 1. What does it store?
#### FaceITBot stores the following: 
##### • User IDs and FaceIT Names (only by using .faceitsave)
##### • It does NOT save anything by using every other command

### 2. Who can access the data?
##### • The Bot developer (me)

### 3. How can I get rid of the data stored?
#### To clear your UserID and your FaceIT name from the database you can use the .faceitdelete command to fully delete ID and name.

### 4. Why does it store the data?
#### It stores User IDs and FaceIT Names because without it, the function to save names for more comfortability does not work.
